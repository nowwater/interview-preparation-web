Web

1. 다음 중 Web 의 특징이 아닌 것은?

   Http 프로토콜과 Https 프로토콜이 존재한다.
   웹만으로 프로그램을 구현할 수 있다.
   WEB 3.0 에서는 시멘틱 기술을 이용한다.
   HTML 기반으로 제작하여 최종 앱 배포 시 아이폰, 안드로이드에서 패키징을 처리한다.
   쿠키는 클라이언트에, 세션은 서버에 저장된다.

3. 다음 중 렌더링 방식과 상관이 없는 것은?

   SSR은 서버 사이드에서 렌더링한 HTML을 클라이언트에게 전달해준다.
   CSR은 클라이언트 사이드에서 뷰 렌더링을 담당한다.
   CSR 방식은 인터랙션이 많은 환경에서 비효율적이다.
   SPA는 최초 한 번만 페이지 전체를 로딩하고 이후 데이터만 변경하여 사용할 수 있는 어플리케이션이다.
   SPA는 CSR 방식을 사용한다.

4. 다음 중 MVC 패턴에 대한 설명이 올바르지 않은 것은?

   Model, View, Controller 로 구성된다.
   코드를 재사용할 수 없는 단점이 있다.
   모델은 핵심 비즈니스 로직을 담당해 DB를 관리하는 부분이다.
   뷰는 사용자에게 보여주는 화면이다.
   컨트롤러는 모델과 뷰 사이에서 정보 교환을 할 수 있도록 연결해준다.
