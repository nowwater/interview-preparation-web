1. 다음 설명에 해당하는 것은?


  • 프로세스가 자원을 얻지 못해서 다음 처리를 진행하지 못하는 상태이다.

  • 시스템적으로 한정된 자원을 여러 곳에서 사용하려고 할 때 발생한다.

  • 발생하기 위한 조건이 존재하며 이중 하나라도 만족되지 않으면 이 상황이 발생하지 않는다.

    • 2. 다음 설명에 해당하는 페이지 교체 알고리즘은?

  • 최근에 사용하지 않은 페이지를 가장 먼저 내보내는 알고리즘

  • 사용하지 않은 페이지는 앞으로도 사용하지 않을 확률이 높다.

  • 실제로 사용할 수 있는 페이지 교체 알고리즘 중에서 가장 좋은 방법 중의 하나이다.

    • 3. 다음 설명에 해당하는 CPU 스케줄링 알고리즘은?

  • 각 프로세스에게 일정한 시간의 Time Quantum 만큼 CPU를 할당해주고, 각 프로세스는 돌아가며 주어진 시간동안 작업을 수행한다.

  • Time Quantum이 너무 크면 FCFS 방식과 같아지고, 너무 작으면 Context Switching이 자주 발생하여 오버헤드가 증가한다.

  • CPU 스케줄링 외에도 다양한 스케줄링 방식에 적용될 수 있는 개념이다.

    • 4. 다음 설명에 해당하는 용어의 이름은?

  • 데이터베이스의 상태를 변화시키기 위해 수행하는 하나의 논리적 작업 단위이다.

  • ACID 특징을 갖는다.

  • OOOO 의 성공 여부로 Commit 또는 Rollback을 수행한다.
